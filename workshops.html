Build Your Own Website

HTML CSS

In this workshop, you'll learn to build an online portfolio using HTML and CSS. The Sudo workshops team has tailored this workshop to be interactive and collaborative, allowing you to easily complete and customize the final project with no prior experience.

Be sure to bring your own laptop and download a text editor (such as Sublime Text) in preparation for the event.

×

Introduction to JavaScript

HTML CSS JavaScript

In this web development workshop, you'll take your knowledge of HTML and CSS to a new level. Javascript will be introduced to add a new interactive dimension to your websites. This workshop is tailored for beginners, but experience in basic HTML and CSS would be beneficial.

Be sure to bring your own laptop and download a text editor (such as Sublime Text) in preparation for the event.

×

Python for Beginners

Python

In this workshop, you'll learn the basic concepts of programming in the language Python. This interactive workshop is tailored for beginners, allowing anyone to easily follow along and develop a customizable project. Learn more about Python here.

Be sure to bring your own laptop and download the newest version of Python in preparation for the event.

Creative Computing

Processing Python

In this workshop, we will cover the basics of programming using Processing and Python to draw and animate pictures. This interactive and collaborative workshop will allow you to develop your own designs, with help from Sudo's volunteer mentors. No previous computer programming or artistic experience is required.

Be sure to bring your own laptop and download Processing in preparation for the event.

WordPress Basics

WordPress HTML CSS

In this workshop, we'll cover the basics of developing a website using Wordpress. This interactive and collaborative workshop will allow you to develop your own custom webpage, with help from Sudo's volunteer mentors. No previous programming experience is required.

Be sure to bring your own laptop in preparation for the event.

Designing Your Website

HTML CSS BOOTSTRAP

This workshop will teach you how to build your own website from scratch and how to create custom and modern designs. You will be learning HTML, advanced CSS, and a few Bootstrap elements to make a custom website.

Be sure to bring your own laptop and download a text editor (such as Sublime Text) in preparation for the event.

Exploring Data Science with R

R

This workshop will cover the basics of Data Science using the language R. You will be learning HTML, advanced CSS, and a few Bootstrap elements to make a custom website.

Be sure to bring your own laptop and install R in preparation for the event.
